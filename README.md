# CSS Grid vs Flexbox

This folder contains a simple HTML demo that compares CSS Grid and Flexbox side by side.

## What it shows

- `index.html` renders a dark-themed page with two sections: one built with CSS Grid and one with Flexbox.
- The Grid example uses `repeat(auto-fill, minmax(150px, 1fr))` to create responsive columns.
- The Flexbox example uses wrapping and centering to show how items flow in a flex layout.

## How to use

Open `index.html` in a browser to view the comparison.

## Notes

- The page is self-contained and uses inline CSS.
- It is intended as a quick visual reference for the differences between the two layout systems.
